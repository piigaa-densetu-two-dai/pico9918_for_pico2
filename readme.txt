【壊れても知りません！】PICO9918 v0.4.1 for Pico2

Raspberry Pi Pico2(rp2350)用のpico9918の回路図(pico9918.pdf)とファームウェア(pico9918.uf2)です。
置き換え対象がTMS9918/TMS9118の場合はpico2ボード要改造です。mod.pngの通りにLEDを外してください。
置き換え対象がTMS9928の場合は改造不要です。TP5の結線も不要です。
MSXのPV7(TMS9118)とHX21(TMS9928)で動作しました。
GROMCLKは省略しています。MSXにおいては使われていないようなので問題ありませんがお気を付けください。
個人でDIYする用途以外には使用しないで下さい。これを基板にして配ったり売ったり等は絶対にしないで下さい。
